Workbench Public Swagger

UI Capabilities

  • Generate or reuse the persistent demo API key stored in MongoDB.
  • Generate synthetic hosts inside a CIDR subnet such as 192.0.2.0/24.
  • Display each host IP address, registration state, and group memberships.
  • Show group cards with host IP membership lists for each group.
  • Add new groups from the GUI; defaults are dev, prod, critical, and isolated.
  • Highlight registration or group changes with the latest timestamp and source metadata.
  • Receive live table refreshes when hosts are generated, created, updated, or cleared.
  • Clear host inventory data while keeping the same API key for repeated demos.

Backend API Capabilities

  • Public JWT authentication using a UI-generated API key.
  • Public host listing for external systems consuming inventory data.
  • Public host creation with hostIP, registered, and optional groups.
  • Public host upsert by IP address to flip the registration state.
  • Public endpoints to add or remove an existing host from a group.
  • MongoDB-backed change tracking through fieldChanges.registered and fieldChanges.groups.
  • Internal UI-only routes stay out of the public Swagger document.

Public API Flow

1

Generate or copy the API key from the Workbench.

2

Call POST /public/auth/token with X-API-Key.

3

Use Authorization: Bearer <jwt> for host APIs.

curl -k -X POST https://:8443/public/auth/token \
  -H "X-API-Key: <api-key>"

curl -k https://:8443/public/hosts \
  -H "Authorization: Bearer <jwt>"

curl -k -X PUT https://:8443/public/hosts/192.0.2.44/groups/critical \
  -H "Authorization: Bearer <jwt>"

Host Data Shape

{
  "hostIP": "192.0.2.44",
  "registered": true,
  "groups": ["prod", "critical"]
}

Demo Notes

  • Use Clear Host Data before a new presentation run.
  • The API key survives resets so client scripts do not need to be edited.
  • Open the Workbench beside a terminal to watch registration and group membership changes appear live.
  • Swagger is intentionally limited to public integration endpoints.
